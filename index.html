1.1 Emacs

I have long been a fan of linux since 2004 or so when I used a distro called Xandros to save my wife's laptop. Since then, I have tried about every version of linux that exists. (Emacs)[www.gnu.org/s/emacs] is an old program that can do powerful things with text. It is used for programming but my main use for it right now will be writing this lonely outpost on the internet. Org-Mode is a mode that runs inside Emacs.
